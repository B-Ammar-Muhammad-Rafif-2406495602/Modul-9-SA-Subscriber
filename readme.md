## what is amqp

amqp is a standard protocol for sending messages between programs. it defines how messages are formatted and transferred between a publisher, a message broker (like rabbit mq), and a subscriber

## What does it mean? guest:guest@localhost:5672 , what is the first guest, and what is the second guest, and what is localhost:5672 is for?

first guest = the username to login to rabbitmq
second guest = the password
local host = rabbitmq is running on your own machine
5672 = the port rabbitmq listens on for incoming ocnnections
